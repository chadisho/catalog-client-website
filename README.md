# Chadisho Catalog (Next.js)

This project is migrated from Vite/React Router to **Next.js App Router** with SSR-ready routing while keeping feature-based architecture under `src/features` and shared concerns under `src/core`.

## Scripts

- `npm run dev` - Start development server
- `npm run build` - Production build
- `npm run start` - Start production server
- `npm run type-check` - TypeScript type check
